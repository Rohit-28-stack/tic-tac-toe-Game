# Tic Tac Toe 

This is a simple Tic Tac Toe game I built using HTML, CSS and JavaScript.
I tried to make it look a bit modern with neon colors and added a small blast animation when someone wins.

# Live Demo
https://Rohit-28-stack.github.io/tic-tac-toe-Game/

# What it does
* Two players can play (X and O)
* Shows the winner correctly
* Reset and New Game buttons
* Small celebration effect when someone wins
* Works on mobile as well


# Tech Used
* HTML
* CSS
* JavaScript

# How to run
Just download the project and open `index.html` in your browser.

Or clone it:
```bash
git clone https://github.com/Rohit-28-stack/tic-tac-toe-Game.git
```
